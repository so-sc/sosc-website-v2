## About

Notes App is a simple, lightweight note-taking application designed for quick and easy note management. Built with React, it provides a clean and intuitive interface for organizing your thoughts.

## Features

- Create, edit, and delete notes
- Search functionality
- Local storage persistence
- Responsive design
- Dark mode support

## Getting Started

```bash
# Clone the repository
git clone https://github.com/so-sc/notes-app.git

# Install dependencies
npm install

# Start the app
npm start
```
