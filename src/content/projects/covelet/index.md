# Covelet

**Covelet** is a secure memory-sharing and digital scrapbooking application built for private groups to preserve and relive their most precious moments.  
It combines private coves, scheduled time capsules, and discovery-focused memory experiences in a mobile-first package.

---

## Key Features

- **Private Coves** – Curated group spaces for sharing memories with trusted members
- **Time Capsules** – Create scheduled memory drops that unlock later
- **Secure Data Flow** – Server-authoritative backend with cascade deletion support
- **Memory Discovery** – Flashbacks and roulette-style reliving features
- **Expo Mobile Experience** – Cross-platform app with native Android support

---

## Tech Stack

Covelet is built using the following technologies:

- **React Native** – Mobile UI framework
- **Expo** – App development and deployment tooling
- **Firebase** – Authentication, Firestore, and notifications
- **Node.js** – Backend runtime
- **Express** – API server framework
- **Cron jobs** – Scheduled notification delivery and background task timing

---

## Project Goals

- Build a private, secure place for group memory sharing
- Enable collaborative time capsule creation and scheduled reliving
- Provide a mobile-first experience with responsive native UI
- Use Firebase and server-side rules for data safety and integrity
